# 🤖 O-Bot — Discord AI Assistant (Groq Powered)

An AI-powered Discord bot built using Groq API and Discord Slash Commands.

## 🚀 Features
- ⚡ Ultra-fast AI responses (Groq API)
- 💬 Slash command `/obot`
- 📏 Handles long responses (auto-splitting)
- 🔐 Secure environment variables
- 🧠 Smart AI assistant behavior

---

## 🛠️ Tech Stack
- Python
- discord.py
- Groq API
- dotenv
